# Teknik Servis & Ürün Takip Sistemi

### Proje Hakkında 
.NET C# ile Devexpress Güçlü Arayüzü Oluşturulmuş Masaüstü Projesi , Müşteri için Ürün Takibi,Genel Bilgi Alma ve İletişim için Asp.Net ile  geliştirilen Web Sayfası,Her İki Projenin Ortak Kullandığı MSSQL Veri Tabanı kullanılıp.DB  First yaklaşımı ile ORM Çatısı Altında Entity Framework kullanılmıştır.

Web Kodları için -> [Web Repository](https://github.com/hamdidamar/TeknikServis-Web)
